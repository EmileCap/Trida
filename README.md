# Projet Base de Données — Application e-commerce

Projet réalisé dans le cadre du BUT Informatique.

Cette application permet de gérer une base de données e-commerce avec une interface utilisateur simple, permettant de :

- Ajouter, modifier et supprimer des articles
- Effectuer automatiquement des calculs sur les données avec SQL
- Naviguer facilement grâce à des listes déroulantes et une interface web

---

## Contribution

- Conception de la base de données (tables, relations, contraintes)
- Développement des requêtes SQL pour gérer les données
- Création de l'interface utilisateur avec Python et le framework Flask
- Tests et validation des fonctionnalités

---

## Technologies utilisées

- SQL (MySQL / PostgreSQL / SQLite selon le projet)
- Python avec Flask pour l’interface web
- Git / GitHub pour le versioning

---

## Compétences développées

- Modélisation et gestion d'une base de données relationnelle
- Développement d'une interface web pour manipuler les données
- Utilisation de SQL pour les opérations et calculs sur les données
- Organisation et gestion d'un projet complet
- Travail collaboratif et gestion de versions avec Git

---

## Objectif académique

L'objectif de ce projet était de se familiariser avec la **modélisation et la gestion d'une base de données relationnelle**, ainsi qu'avec le développement d'une interface web permettant d'interagir avec les données.  
Ce projet m'a permis de mettre en pratique les concepts appris en cours, de comprendre la logique des bases de données et de développer mes compétences en développement web avec Python et Flask.
